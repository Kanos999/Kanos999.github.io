About the project

This project aims to showcase the capabilities of advanced computer vision technology. This interactive program utilizes a standard webcam to detect and track the motion of a ping pong ball during play, accurately counting each bounce in real time. Designed to demonstrate the integration of real-time image processing, this project not only highlights the precision of object tracking but also showcases potential applications in sports technology and training tools.

Technologies Used
